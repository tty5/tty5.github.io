# 记录task执行, 被换出去过程
python
```
#!/usr/bin/python

import time

time.sleep(1)
time.sleep(1)
time.sleep(1)
time.sleep(1)
time.sleep(1)
time.sleep(1)
```

## cmd
```
perf record -e sched:sched_switch  -a -- python 1.py
```

## 如果需要知道每次睡眠的原因的话, 调用栈就是原因, 可以把唤醒的调用栈也加上
```
perf record -e sched:sched_switch  -a --call-graph dwarf -- python 1.py
```
