# ubuntu mirror
```
sed -i s@archive.ubuntu.com@mirrors.163.com@g /etc/apt/sources.list
sed -i s@security.ubuntu.com@mirrors.163.com@g /etc/apt/sources.list
apt update
apt install bash-completion -y
source  /etc/bash_completion
```
